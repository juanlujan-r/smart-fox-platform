# Smart Fox - Lógica HR & Automatización
Contexto: Next.js 15 Server Actions, XLSX Library.

## 1. Importación Masiva
- Implementar función para procesar archivos Excel/CSV.
- Mapear columnas de Excel a los campos de la tabla `profiles`.
- Validar duplicados por `document_id` antes de insertar.

## 2. Generador de Certificados
- Crear una Server Action que use la información de la DB para llenar una plantilla de certificado laboral.
- Campos requeridos en la plantilla: Nombre Completo, ID, Tipo de Contrato, Fecha de Ingreso y Cargo (role).
